# BreakLingo Color Palette

> Source of truth for all design-system colors.
> All values are defined as CSS custom properties in `src/index.css` using HSL format.

| Class | Hex | Usage |
|-------|-----|-------|
| `text-green-500` | `#22C55E` | Active recording mic indicator |
| `text-green-600` | `#16A34A` | Listening status, correct answers |
| `bg-green-500` | `#22C55E` | Recording mic button background |
| `bg-green-400` | `#4ADE80` | Listening ping animation |
| `text-yellow-500` | `#EAB308` | Warning icon, star icon |
| `text-yellow-600` | `#CA8A04` | Processing status, suggestion feedback |
| `text-blue-500` | `#3B82F6` | Speaking status volume icon |
| `text-blue-600` | `#2563EB` | Speaking status label |
| `text-red-600` | `#DC2626` | Incorrect answers, error feedback |

|---------|-----------|------|-------|
